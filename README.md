# Coffee Corner ☕

A responsive coffee shop website built with React. This is a student project for CSci390 Web Programming course.

## Live Demo
🔗 [View Live Website](https://coffee-corner.vercel.app)

## GitHub Repository
🔗 [View Source Code](https://github.com/42330626-lgtm/coffee-corner)

## Features
- ✅ 4 pages: Home, Menu, Contact, About
- ✅ Responsive design (works on desktop, tablet, and mobile)
- ✅ Hamburger menu on small screens
- ✅ Contact form with validation (checks empty fields and valid email)
- ✅ 9 menu items with prices
- ✅ Contact information (Beirut, +961 78 746 783)
